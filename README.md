## Development
* **Platform**: Windows 10 32 Bit and 64 Bit
* **Programming Language**: ISO C++ Latest Draft Standard (> C++17)
* **Compiler**: MSVC++ 16.0.1
* **IDE**: Visual Studio 2019

## Features

* Algebras:
  * Complex
  * Dual
  * Hyperbolic
* Compile-time math (*work in progress*)
* SIMD (*work in progress*)
