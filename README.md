## Development
* **Platform**: (minimum) Windows 8.1 32 Bit and 64 Bit
* **Engine Programming Language**: ISO C++ Latest Draft Standard (> C++17)
* **Compiler**: MSVC++ 15.0
* **IDE**: Visual Studio 2017

## Features

* Complex Numbers
* Dual Numbers
* Hyperbolic Numbers
* SIMD (*work in progress*)
